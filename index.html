DoeVest

Sistema de Doação de Roupas

Sobre o Projeto

Aplicativo desenvolvido em Flutter para auxiliar no cadastro e gerenciamento de roupas destinadas à doação.

Funcionalidades

  • Login de usuários
  • Cadastro de roupas
  • Listagem de roupas cadastradas
  • Remoção de roupas

Tecnologias Utilizadas

  • Flutter
  • Dart
  • Firebase Authentication
  • SQLite
